The Core Concept

Understanding what inverse functions truly represent in mathematics.

The "Undoing" Action

An inverse function, denoted as f⁻¹(x), describes a function's ability to "undo" the operation of another function, f(x). If a function f takes an input a and produces an output b, then its inverse function f⁻¹ will take the input b and produce the original input a.

This relationship is expressed as: if f(a) = b, then f⁻¹(b) = a

This "undoing" action is formally expressed through function composition, where f(f⁻¹(x)) = x and f⁻¹(f(x)) = x.

Machine Analogy

Think of a machine that multiplies by 3 then adds 2. Input 4 becomes 14. The inverse machine would subtract 2 then divide by 3, turning 14 back into 4.

f 4 → 3×4+2 = 14
f⁻¹ 14 → (14-2)÷3 = 4

Input-Output Relationship

Original Function

Input x → Output y

f(x) = y

Inverse Function

Input y → Output x

f⁻¹(y) = x

Domain & Range

Domain and range are swapped.

The Prerequisite: One-to-One Functions

A function must be one-to-one to have an inverse. A function is one-to-one if each output value corresponds to exactly one input value. We can use the horizontal line test to determine this.

Horizontal Line Test

A function is one-to-one if and only if no horizontal line intersects its graph more than once.

For example, f(x) = x² fails the test because y = 4 intersects at x = 2 and x = -2. However, restricting the domain to x ≥ 0 makes it one-to-one with inverse f⁻¹(x) = √x.

Why Non-One-to-One Functions Fail

If f(x₁) = y and f(x₂) = y, then f⁻¹(y) would need to equal both x₁ and x₂

This violates the definition of a function (one input → one output)

One-to-one functions ensure unique inverse mappings

The Partner Requirement: Onto (Surjective) Functions

For a function to be invertible it must also be onto (surjective). A function f : A → B is onto if every element in the codomain B is actually “hit” by some element from the domain A.

Vertical Line Test for Range

A function is onto if and only if every horizontal line drawn through the codomain intersects the graph at least once.

For example, g(x) = eˣ maps ℝ → ℝ is not onto because the negative numbers in the codomain are never reached. Restrict the codomain to (0, ∞) and it becomes onto, hence invertible; the inverse is g⁻¹(x) = ln x.

Why Non-Onto Functions Fail

If y ∈ B is never produced by f, then f⁻¹(y) is undefined—no input to send back to

The alleged inverse would have a “hole” in its domain, violating the function definition

Onto functions guarantee the inverse’s domain is the entire intended codomain

The Step-by-Step Process

Step Action Description
1 Replace f(x) with y Notational change to make manipulation easier

3 Solve for y Use algebraic techniques to isolate y
4 Replace y with f⁻¹(x) Final notation indicating inverse function

Worked Examples

1 Linear Function

f(x) = 2x + 3
y = 2x + 3
x = 2y + 3
y = (x - 3)/2
f⁻¹(x) = (x - 3)/2

2 Rational Function

f(x) = 1/x + 4
y = 1/x + 4
x = 1/y + 4
y = 1/(x - 4)
f⁻¹(x) = 1/(x - 4)

Verifying Inverses Compositionally

To verify that two functions are inverses, we use the fundamental property: f(f⁻¹(x)) = x and f⁻¹(f(x)) = x.

Verification for Linear Example:
f(f⁻¹(x)) = f((x-3)/2) = 2((x-3)/2) + 3 = x
f⁻¹(f(x)) = f⁻¹(2x+3) = ((2x+3)-3)/2 = x

✓ Both compositions yield x, confirming inverse relationship

Graphical Interpretation

Discover the elegant geometric relationship between functions and their inverses through visual symmetry

The Reflection Principle

The graph of a function and its inverse are reflections of each other across the line y = x. If you fold the coordinate plane along y = x, the graphs would perfectly overlap.

Point Mapping

Every point (a, b) on f becomes (b, a) on f⁻¹

Symmetry Axis

y = x is the perpendicular bisector connecting (a,b) to (b,a)

Identity Connection

y = x represents the identity function I(x) = x

The Coordinate Swap Mechanism

Geometric Interpretation

Swapping x and y coordinates is the algebraic equivalent of reflecting across y = x. The midpoint between (a, b) and (b, a) is ((a+b)/2, (b+a)/2), which lies on y = x, confirming it as the perpendicular bisector and axis of reflection.

Plotting Functions and Their Inverses

Visual Verification

Graphing both functions on the same axes reveals the mirror image effect, providing immediate visual confirmation of the inverse relationship. This technique is particularly valuable for understanding complex functions where algebraic verification might be challenging.
